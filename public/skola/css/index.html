CSS

CSS je jazyk, který používáme ke stylování dokumentu HTML a popisuje, jak by se měly jeho prvky zobrazovat. Je to základem moderního designu.

Třídy, pseudotřídy a identifikátory

Třídy, pseudotřídy a identifikátory jsou základními stavebními kameny selektorů v CSS. Třídy (.) umožňují opakovaně aplikovat stejné styly na různé prvky, zatímco identifikátory (#) jsou unikátní pro každý prvek na stránce. Pseudotřídy (:) poskytují pokročilé možnosti stylování dynamických stavů prvků, jako je například změna vzhledu odkazu při najetí myší.

Box model

Box model je základní koncept v CSS, který určuje, jak se vykreslují prvky na webové stránce. Skládá se z obsahu, vnitřního okraje (padding), rámečku (border) a vnějšího okraje (margin). Pochopení box modelu je klíčové pro správné rozvržení a stylování webových stránek.

content-box
border-box

Flexbox

Flexbox je moderní CSS layoutový modul, který usnadňuje uspořádání prvků v kontejnerech s dynamickou velikostí. Umožňuje snadno vytvářet responzivní a flexibilní designy, které se přizpůsobují různým velikostem obrazovek. Práce s layoutem je rychlá a intuitivní.

Animace

Animace v CSS umožňují přidávat pohyb a interaktivitu na webové stránky bez potřeby JavaScriptu. Pomocí přechodů (transitions) a klíčových snímků (keyframes) lze snadno vytvářet efekty jako změny barvy, posouvání nebo otáčení prvků. Tyto techniky zvyšují vizuální atraktivitu a uživatelskou přívětivost webu.

Proměnné

Proměnné v CSS umožňují definovat opakovaně použitelné hodnoty, které usnadňují správu a údržbu stylů. Definují se pomocí -- a lze je jednoduše použít pomocí funkce var(). Díky proměnným můžete snadno měnit témata nebo globální styly na celém webu s minimálním úsilím.
